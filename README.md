#### 开始之前
- 安装npm
  - 方式一：通过安装nodejs LTS，自动安装上npm
  - 方式二：通过nvm

#### npm i <packageName>
- 全局安装 -g 或 --global

#### npm publish
- 创建需要发布的包的文件夹
  - npm init 为npm包的进行初始化
  - npm init --yes 或 npm init -y 使用默认信息作为package.json的内容

#### 一些你可能会用到的npm命令
- npm init 初始化一个npm包
- npm run 
- npm run env
- npm list 或 npm ls
- npm ll <packageName>
